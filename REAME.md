### How to Start Project with git

```shell
$ mkdir first-repo
$ cd first-repo
$ git init
$ touch REAME.md
$ git add REAME.md
$ git commit

$ git remote add origin {repo addr}
$ git branch -M main
$ git push -u origin main
```
